# Portfólio Pessoal

Meu portfólio pessoal desenvolvido com HTML e CSS, criado para apresentar minha trajetória acadêmica, interesses na área de tecnologia e disponibilizar formas de contato profissional.

## Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de centralizar minhas informações profissionais e servir como um espaço para apresentar minha evolução na área de desenvolvimento de software.

No site é possível encontrar:

- Informações sobre minha formação acadêmica;
- Tecnologias e áreas de interesse;
- Links para meu LinkedIn e GitHub;
- Formas de contato.

## Tecnologias Utilizadas

- HTML5
- CSS3
